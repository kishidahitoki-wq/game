# ゲーム仕様書 (Ver 0.1)

## 1. 概要
- **ジャンル:** 2D横スクロールアクション・エンドレスランナー
- **コンセプト:** ネコが走りながら障害物を避け、穴を飛び越える
- **目的:** HPが0になるか、穴に落ちるまでできるだけ長距離を走り続ける（スコア/走行距離を競う）

## 2. プレイヤー（ネコ）の挙動
- **基本動作:** 自動で右方向に走り続ける。
- **操作方法（矢印キー）:**
  - `[↑]`: ジャンプ（障害物を飛び越える）
  - `[↓]`: しゃがむ（上からの障害物をくぐる）
  - `[→]`: 右へ移動（加速）
  - `[←]`: 左へ移動（減速）

## 3. ゲームシステム
- **ライフ（HP）制:**
  - プレイヤーはHPを持つ。
  - 障害物にぶつかるとHPが減少する。
  - HPが0になるとゲームオーバーとなる。

## 4. 登場オブジェクト
- **地形ギミック:**
  - 坂道（上り坂、下り坂）
  - 穴（落ちると即ゲームオーバー）
- **障害物の例:**
  - ゴミ箱
  - 犬、走ってくる車（タイミングよく避ける）
  - カラス（しゃがんで避ける）
- **アイテムの例:**
  - 魚、コイン（取得するとスコアアップ）
## 5. 技術スタック
- **プラットフォーム:** Webブラウザ
- **フロントエンド:** HTML5, CSS3, JavaScript (Canvas API などを想定)
