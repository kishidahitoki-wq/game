# ゲーム仕様書 (Ver 0.1)

## 1. 概要
- **ジャンル:** 2D横スクロールアクション・エンドレスランナー
- **コンセプト:** ネコが走りながら障害物を避ける
- **目的:** HPが0になるまでできるだけ長距離を走り続ける（スコア/走行距離を競う）

## 2. プレイヤー（ネコ）の挙動
- **基本動作:** 自動で右方向に走り続ける。
- **操作方法（矢印キー）:**
  - `[↑]`: ジャンプ（障害物を飛び越える）
  - `[↓]`: しゃがむ（上からの障害物をくぐる）
  - `[→]`: 加速
  - `[←]`: 減速

## 3. ゲームシステム
- **ライフ（HP）制:**
  - プレイヤーはHPを持つ。
  - 障害物にぶつかるとHPが減少する。
  - HPが0になるとゲームオーバーとなる。

## 4. 登場オブジェクト
- **ギミック（障害物）の例:**
  - ゴミ箱、土管、水たまり（ジャンプで避ける）
  - 犬、走ってくる車（タイミングよく避ける）
  - カラス（しゃがんで避ける）
- **アイテムの例:**
  - 魚、コイン（取得するとスコアアップ）
## 5. 技術スタック
- **プラットフォーム:** Webブラウザ
- **フロントエンド:** HTML5, CSS3, JavaScript (Canvas API などを想定)
